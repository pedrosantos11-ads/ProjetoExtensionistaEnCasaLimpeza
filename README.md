# EnCasaLimpeza 🧹

Projeto extensionista — site para empresa de limpeza doméstica.

## Tecnologias
- Python + Flask (backend)
- HTML/CSS/JS (frontend)

## Como rodar
```bash
source venv/bin/activate
python3 app.py
```
Acesse: http://127.0.0.1:5000
